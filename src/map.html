Lösningar

I denna uppgift har jag användt ett externt API för att bifoga en karta på webbplatsen. Lösningen jag valde var att kombinera Nominatim API för att få fram kordinater(latitude och longitude) med OpenStreetMap som eninbäddad iframe.

Krav på uppgiften:

  • Det ska finnas ett textfält någonstans på sidan där användaren kan skriva in en plats.
  • Vid val av plats ska FetchAPI-anrop göras tillsammans med async/await.
  • Platsen ska visuellt visas på kartan med markör för vald plats.

För att skapa en liknande lösning hittar du länkar till documentation och introduktion i den här sidans footer.
